Design an OER Module Project:

Beginner

This project was designed to help you gain familiarity with adding OER Schema to a project. Typically, a project will have a longer description.

Readings:

Collins, M. (2016, May 27). Interoperable Open Education Resources (OER) [Web log post]. Retrieved September 27, 2016, from http://www.collinsartanddesign.com/blog/2016/5/27/interoperable-open-education-resources-oer

Wehr, K. M. (2016, June 15). Is OER Really Open? [Web log post]. Retrieved September 27, 2016, from https://medium.com/@katrina.m.wehr/is-oer-really-open-1ac98e0913d8

Steps to Completion

  1. Read the readings.
  2. Create the project.
  3. Submit the project.

Learning Objectives

  • Learn to use the SEED OER builder interface by constructing a learning module.
  • Gain basic understanding of how learning content can be remixed when structured with OER Schema.
